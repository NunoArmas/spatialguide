What is Spatial Guide?


It is an audio guiding app for Android that tracks current location and gives visual and audio info on points of interest in tour routes.
The app aims to deliver a more immersive feeling using spatial audio effects to give the impression that the points of interest are talking with the tourist. This effect is achieved by a procedure called Auralization, which simulates the audible acoustic sceneries.



We are going to use a IMU (Inertia Measurment Unit) to track the user's head direction.



Finally, using the GPS, the system tracks the tourist location and if the user is inside the trigger area, then the application will display the visual information on the smartphone's screen and combining the location and orientation, it will reproduce a stereo sound with the illusion that it is coming from the point of interest itself.
